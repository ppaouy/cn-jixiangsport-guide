# cn-jixiangsport-guide

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内包含若干自行编写或整理的静态页面，内容涉及信息展示、工具页面、参考文档等，不针对任何特定域名或网站。所有页面均为独立 HTML 文件，可直接在浏览器中打开使用。

## 目录说明

- `docs/` — 存放主要 HTML 页面文件的目录
- `assets/` — 页面引用的资源文件（如样式表、图片、脚本等）
- `archive/` — 历史版本或不再维护的页面归档
- `README.md` — 本文件

页面文件名通常体现其大致用途，例如 `guide.html`、`tools.html` 等。具体内容请浏览 `docs/` 目录下的文件。

## 页面归档说明

本仓库中的 HTML 页面均为独立文件，不依赖外部服务或数据库。每个页面在创建时已尽可能保持自包含，以便离线查看或简单部署。

- 页面之间的链接可能指向仓库内其他文件，也可能指向外部参考资源。
- 部分页面可能包含引用来源或数据说明，均已在页面内标注。
- 归档页面原则上不再更新，但会保留在 `archive/` 目录下供参考。

## 维护说明

- 本仓库不定期更新，新增页面或修正内容。
- 欢迎提交 Issue 指出页面中的错误或改进建议。
- 暂不接受直接修改仓库内容的 Pull Request，如有需要请先通过 Issue 沟通。
- 仓库维护者保留对页面内容进行增删修改的权利。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 打开 `docs/` 目录下的 HTML 文件即可浏览。
3. 也可将 `docs/` 目录部署到任意静态服务器上使用。

## 许可

本仓库内容仅供个人学习与参考使用。具体页面若有额外声明，以页面内标注为准。
